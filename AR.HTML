O que é Hypertext Markup Language (HTML)

O HTML é uma linguagem de marcação utilizada para estruturar páginas na web.
Ele define elementos como títulos, parágrafos, imagens e links.

Como e porquê surgiu o HTML

O HTML surgiu com o objetivo de compartilhar documentos na internet de forma organizada.
Foi criado por Tim Berners-Lee para facilitar a comunicação entre pesquisadores.

O que é eXtensible Hypertext Markup Language (XHTML)

O XHTML é uma versão mais rigorosa do HTML baseada em XML.
Ele exige que o código seja bem estruturado e siga regras mais rígidas.

Qual a diferença de XHTML para HTML

O XHTML exige regras mais rígidas, como fechamento de todas as tags e escrita correta.
Já o HTML é mais flexível e tolerante a erros.

O que é eXtensible Markup Language (XML)

O XML é uma linguagem de marcação usada para armazenar e transportar dados.
Diferente do HTML, ele não define como os dados serão exibidos.

Qual a diferença de HTML para XML

O HTML é usado para exibir conteúdo em páginas web, enquanto o XML organiza e armazena dados.
O HTML foca na apresentação e o XML na estrutura da informação.
